Web APIs & HTTP

What is HTTP?

HTTP stands for Hypertext Transfer Protocol, and is used to structure requests and responses over the internet. HTTP requires that data is transferred from one point to another over the network.

What is TCP?

TCP (Transmission Control Protocol) is what actually handles the transfer of data. TCP is used to manage many types of internet connections in which one computer wants to send data to another.

How it all works together

When you type an address into the browser to load a webpage, you command TCP to open a connection between your browser (the client) and the server to which the url relates to. Once the TCP connection is established, the client sends a HTTP GET request to request the webpage. With HTTP/1.0 (the first version of HTTP), once the browser sends back a response, the TCP connection is closed.
