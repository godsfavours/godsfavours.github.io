Relative Measurements

em

em represents the font-size of the current element. If a font-size is not explicitly set, it uses the browser's default size.

This paragraph is 1em (16px font-size)

This paragraph is 2em (32px)

This paragraph is 3em (48px)

rem

rem stands for root em. It checks the root (<html>) element for text size and ignores that of the current element.

This paragraph is 1rem (18px default browser font-size)

This paragraph is 2rem (36px)

This paragraph is 3rem (54px)

Percentages: Height and Width

Percentages are used to set dimensions of child elements relative to their parent elements. Height percentage is set relative to high of parent container, as well as width

Outter box is 300x300, and inner box's dimensions are set as percentages (50%x75%)

Percentages: Padding and Margin

Percentages can also be used to set padding and margin. Unlike height and width, using percentages to set padding or margin is always only relative to the parent's width.

Outter box is 300x350, and 50% margin is applied to inner box, giving it 150px margin
