Justify Items

Use justify-items on grid containers to position grid items within their columns. This causes the grid item to use up only the space it needs, rather than the default stretching

Grid Auto Rows and Grid Auto Columns

grid-auto-rows and grid-auto-columns define the size of implicitly added rows/columns. CSS automatically (implicitly) adds new rows and/or columns when more grid items are added to the grid than can be fit using the explicitly defined parameters (for example, grid-template-row explicitly defines the grid to have a set number of rows with the given lengths)

Grid Auto Flow

grid-auto-flow specifies whether implicitly added items are added to rows (left to right, new row if needed) or columns (top to bottom, with a new column if needed). The default grid-auto-flow is row.
