Media Queries

Media queries can be used to change the style of html elements based on the screen width.

Use media queries to apply specific css styles depending on screen size.

resize the browser and notice how the width of the box changes.

Dots Per Inches (DPI)

Use DPI to change style based on screen resolution. For example, to render a high quality images to users who can support it.
